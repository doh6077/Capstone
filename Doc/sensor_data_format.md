**This documentation is for tracking API data format**

## Sensor Metadata Format:

```json
{
    "macAddress": "34:b7:da:5d:80:0c",
    "trigerPin1": 7,
    "echoPin1": 8,
    "trigerPin2": 12,
    "echoPin2": 4,
    "trigerPin3": 2,
    "echoPin3": 13
}
```

## Sensor Reading Data Format:

```json
{
    "macAddress": "34:b7:da:5d:80:0c",
    "trigerPin": 7,
    "echoPin": 8,
    "data": 20.6
}
```
